# 墨缚之塔 v0.2.0

发布日期：2026-07-23

## 交付内容

- 60U × 30U 连续卷轴战场与 1/256U 定点坐标。
- 1920 × 960 连续控制场、960 × 480 高精度底图、确定性笔刷、浸润、墨井、结晶、精研、噤声和净化区域。
- 书灵逻辑仍按原格点结算，表现改为柔弧、S 形扫笔与自然流线；移动和涂色共用平滑曲线，笔刷角度连续跟随曲线法线。
- 连续笔刷加入毛笔笔压：起笔、收笔缩至约 22%，按线性包络均匀增粗和变细，最粗点由曲线内部曲率峰值决定。
- 笔刷速率改为单笔刷线性缩放：1 为约 1U、2 为 2U、3 为 3U，不再生成左右副笔刷；茜娜、菲涅、20735 与三个初始模板使用不同形状。
- terrain、Spine、overlay 三层同步画布，支持缩放、拖动、DPR 和窄屏布局。
- 六个初始书灵、连续移动播放、攻击/起飞/降落动画切换和路径同步落墨。
- 51 张卡牌、3 个角色技能、33 个书灵笔刷及空间效果配置。
- 20735 的归档坐标快照、奇点序列、无人机协同、冷却与工事规则。
- 完整保留 v0.1.0 的角色选择、构筑、房间、HUD、手牌、AI、回合、奖励与胜负流程。

## Spine 资源范围

本版本的 33 个书灵玩法模板均使用同名独立 Spine 资源包。每套资源包含 Spine 4.2.43 骨骼、atlas、纹理和预览图；基础动作直接使用各自导出，专属的起飞、飞行、降落、能力、护盾和停用动作按模板能力接入。对不具备某项专属动作的模板，运行时仍保留基础动作降级映射。
